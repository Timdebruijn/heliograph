Heliograph — web installer

Flash the latest firmware straight from your browser. No tools to install.

Before you start
  1. Use Chrome or Edge on a computer (WebSerial does not work in Firefox/Safari, or on phones).
  2. Connect the ESP32-S3 board to this computer over USB.
  3. Click Connect below and pick the serial port that appears.
Install

This writes a full factory image: a board that was already set up will lose its settings (WiFi, passwords) and start over at the setup portal. Updating an already configured bridge is easier over the air: open its web page and use Settings → Firmware update with the matching heliograph-*-<board>.bin from the latest release.

After flashing: the device starts an access point named Heliograph-Setup-XXXX. Join it and the setup page opens by itself.
